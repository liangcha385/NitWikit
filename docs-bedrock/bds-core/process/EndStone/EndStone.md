# EndStone

EndStone 提供类似 Bukkit 插件开发者的友好 API，简化了低层系统 API 和挂钩的处理。可以使用 C++ 或 Python 编写插件。

EndStone 的很多插件可以在 **minebbs** 或者 **PyPI** 上查找了，详情请看 [EndStone安装教程](EndStone-plugins.md)。
